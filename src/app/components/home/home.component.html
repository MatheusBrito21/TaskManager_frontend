TaskManager Sistema gerenciador de tarefas

Olá! Seja bem vindo ao TaskManager desenvolvido por mim no para a empresa NL Informática.
Esse é um sistema para gerenciamento de tarefas simulando uma simples listagem de atividades desenvolvida em Angular e Spring Boot.

O Projeto tem como objetivo principal apresentar a aplicar ALGUMA COISA, Autenticação e Autorização com Spring Security e muito mais. O Código fonte do projeto pode ser acessado através do link do Github no menu lateral.

Fique a vontade para testar o sistema e deixar sua sugestão.
