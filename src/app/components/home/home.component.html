TaskManager Sistema gerenciador de tarefas

Olá! Seja bem vindo ao TaskManager desenvolvido por mim no para a empresa NL Informática.
Esse é um sistema para gerenciamento de tarefas simulando uma simples listagem de atividades desenvolvida em Angular e Spring Boot.

O Projeto tem como objetivo principal demonstrar minhas habilidades e conhecimentos técnicos para cosntrução de uma API Rest do zero, trazendo soluções simples através da integração de várias tecnologias.
